# python_repeat

ProgateでPython繰り返し編まで学んだ内容を応用して、プログラムを2つ作成しました。
学習記録リポジトリです。  
辞書・for文・while文・if文を用いた簡単なアプリケーションを作成しています。

---

## 📌 ファイル構成

### 1. inventory_management.py
簡易在庫管理プログラム。

- 辞書で商品と価格を管理
- for文で商品を順番に処理
- ユーザー入力に応じて購入個数を取得
- 所持金と比較して購入可否を判定
- 残金を表示

### 学習ポイント
- 辞書のキーと値の扱い
- 型変換（int / str）
- 条件分岐（if）
- breakによるループ終了制御

---

### 2. household_account_book.py
簡易家計簿アプリ。

- 予算（budgets）と支出（spents）を辞書で管理
- while文で複数回の入力を受け付け
- カテゴリ追加機能あり
- 各カテゴリの合計支出を表示
- 総支出の計算
- 予算オーバー判定
- 一番使ったカテゴリの表示

### 学習ポイント
- 入れ子のwhile文
- 辞書へのキー追加
- 合計値の計算
- 最大値の更新ロジック
- ユーザー入力の検証

---

## 🎯 目的

Pythonの基礎構文の理解を深めるため、  
実用的なミニアプリを自作することを目的としています。
